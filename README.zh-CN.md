# Symphony 中文说明

Symphony 会把项目工作拆分为隔离、自治的实现运行，让团队把重点放在管理工作本身，而不是持续监督编码代理。

本仓库当前包含：

- `SPEC.md`：项目规范说明。
- `README.md`：英文版项目简介。
- `elixir/`：实验性的 Elixir 参考实现。

## 快速了解

Symphony 适合已经采用 harness engineering 工作方式的代码库。它进一步把“管理编码代理”提升为“管理待完成的工作”。

如果你想快速开始，可以优先查看以下文件：

- `README.md`
- `SPEC.md`
- `elixir/README.md`
